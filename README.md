# dotfiles
This is personal vim and zsh config.

### setup
```
$ cd && ln -s dotfiles/.vim
$ mkdir -p ~/.vim/bundle
$ git clone https://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim
$ open-some-file-with-vim-and-install-vim-plugins
$ mkdir -p ~/.vim/colors
$ cp ~/.vim/bundle/vim-railscasts-theme/colors/railscasts.vim ~/.vim/colors/
```
